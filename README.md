# React Movie App

It's a simple movie app using React.js

Checkout the demo [here](https://seacinema.vercel.app/).

## Features

- [x] Login
- [x] Register
- [x] Movie list
- [x] Movie detail
- [x] Ticket booking
- [x] Transaction detail
- [x] Top up balance
- [x] Withdraw balance
- [x] Search movie by title
- [x] Transaction history
- [ ] Cancel order/ticket
- [ ] Edit user profile
- [ ] Usefull documentation
